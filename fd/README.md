# 页面

## 注册

乳白色背景，中间容器像一个方形的坑一样下陷，可点击的文字添加下划线或更换颜色

## 登录

乳白色背景，中间容器像一个方形的坑一样下陷，可点击的文字添加下划线或更换颜色

## 忘记密码

乳白色背景，中间容器像一个方形的坑一样下陷，可点击的文字添加下划线或更换颜色

## 用户信息

可修改用户名、邮箱、密码

## 管理面板

从左到右分三栏，分别是用户、组、权限

- 用户栏展示 ID、用户名、邮箱，默认按 ID 升序排序，可在升序排序和倒序排序间切换，或按照用户名、邮箱排序
- 组栏展示 ID、名称，默认按 ID 升序排序，或按照名称排序
- 权限栏展示 ID、名称，默认按 ID 升序排序，或按照名称排序

每栏最上方有新建元素按钮

淡色显示被禁用的元素

每个元素右侧有两个图标按钮：编辑和删除

用户编辑中可管理组关联
组编辑中可管理用户关联和权限关联
权限编辑中可管理组关联
被禁用的元素依旧可以操作其关联关系

点击元素即选中此元素，每栏只能选择一个元素

点击某个用户后，只显示该用户的组和权限；如果进一步点击了某个组，只显示该组对应权限；或者如果进一步点击了某个权限，只显示包含该权限的组
点击某个组后，只显示该组对应的用户和权限
点击某个权限后，只显示该权限的组合用户；如果进一步点击了某个组，只显示该组对应的用户；或者如果进一步点击了某个用户，只显示包含该用户的组
总结下就是：点击了什么，就获取和选中数据关联的另两栏数据，和现有数据取交集来展示；筛选后无数据时，展示"空"

# 页面跳转

1. 如果从其他页面跳转到登录注册页面，登录注册后跳转回原页面；否则跳转到用户信息页面
2. 根据状态中的 scope 是否有 * 来决定是否显示管理面板
3. 跳转到管理面板之前请求 /verify_access_token，检查返回值的 scope 中是否包含 *，包含 * 才能跳转到管理面板
